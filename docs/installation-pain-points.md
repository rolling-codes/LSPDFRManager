# LSPDFR Installation Pain Points (Research Notes)

This repository update targets recurring setup issues repeatedly seen in LSPDFR support channels:

1. **Game/RPH compatibility mismatches after GTA title updates**
   - Users often hit *"game version not supported"* until RPH updates.  
2. **Incorrect install locations**
   - RAGE Plugin Hook or plugin files are frequently installed outside the GTA V root / Plugins folders.
3. **Missing dependencies**
   - Script/ASI ecosystems can fail without ScriptHookV.
4. **Security tooling interference**
   - Antivirus/BattlEye frequently surfaces in troubleshooting steps.

The install workflow now surfaces preflight warnings for these conditions in-app before install.
