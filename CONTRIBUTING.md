# Contributing to LSPDFR Manager

First off, thank you for considering contributing to LSPDFR Manager! It's people like you that make LSPDFR Manager such a great tool for the GTA V modding community.

## How Can I Contribute?

### Reporting Bugs
* Check the existing issues to see if the bug has already been reported.
* If not, open a new issue, providing as much information as possible, including steps to reproduce the bug and your environment details.

### Suggesting Enhancements
* Open a new issue and describe the enhancement you'd like to see.
* Explain why this enhancement would be useful to other users.

### Pull Requests
* Fork the repository and create your branch from `master`.
* If you've added code that should be tested, add tests.
* Ensure the test suite passes.
* Make sure your code follows the existing style and conventions.
* Write a clear title and description for your pull request.

## Style Guide

* We follow standard C# and .NET coding conventions.
* Use meaningful variable and method names.
* Keep methods small and focused on a single task.
